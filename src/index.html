Face Finder

Face Finder is a web application deployed on render that allows users to detect faces in images using the Clarifai API. Users can register, sign in, and upload images to the app. After submitting an image, the app displays a box around each of the faces it finds. It uses a PostgreSQL database to store user information, and the backend is built with Node.js and Express.

Memento

This React-based memory matching game is deployed on Firebase. Players click on two cards to reveal their images, and if the images match, the cards remain face up. If the images do not match, the cards are flipped back over, and the player must try again.The game continues until all cards have been matched. The app is PWA-ready and includes a badge that tracks the user's wins.
